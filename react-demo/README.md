# React Demo 练习

这是 Day 2 / Day 3 用的 React + Vite 模板。

## 启动

```bash
cd react-demo
npm install
npm run dev
```

打开终端提示的本地地址，一般是：

```text
http://localhost:5173
```

## 你要看懂的文件

```text
src/App.jsx        页面组件
src/App.css        页面样式
src/mockProject.js mock 数据
API-CONTRACT.md    接口约定示例
```

## 练习顺序

1. 修改 `mockProject.js` 里的项目名和功能列表。
2. 在 `App.jsx` 里找到 `Hero`、`FeatureList`、`DemoPanel`、`TeamSection`。
3. 给 `team` 数组新增一个角色。
4. 给 `features` 数组新增一个功能。
5. 用 Git 提交一次。
